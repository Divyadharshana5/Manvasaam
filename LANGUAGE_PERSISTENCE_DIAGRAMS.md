## Summary

The language persistence system flows as follows:

1. **User selects language** â†’ Triggers `setSelectedLanguage()`
2. **Language is saved** â†’ To both localStorage and cookies
3. **Context updates** â†’ All components using `useLanguage()` are notified
4. **Components re-render** â†’ With new language translations
5. **On page navigation** â†’ Language persists via context
6. **On page refresh** â†’ Language restored from localStorage
7. **On browser restart** â†’ Language restored from localStorage/cookies
